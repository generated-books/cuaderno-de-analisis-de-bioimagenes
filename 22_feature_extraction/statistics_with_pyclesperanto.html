Estadísticas básicas con pyclesperanto#

Un caso de uso común para el procesamiento de imágenes en el contexto biológico es derivar estadísticas de objetos segmentados. clEsperanto ofrece una función para eso: statistics_of_labelled_pixels.

import pyclesperanto_prototype as cle

import pandas as pd
from skimage.io import imread, imsave, imshow
import matplotlib
import numpy as np

# initialize GPU
cle.select_device("RTX")

<Apple M1 Max on Platform: Apple (2 refs)>

# load data
image = imread('../../data/blobs.tif')

# segment the image
labels = cle.voronoi_otsu_labeling(image, spot_sigma=3.5)
cle.imshow(labels, labels=True)
